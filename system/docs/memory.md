# Memory Architecture

The Knowledge System itself is a key part of Jarvis's long-term memory. Over time, docs influence memory and reasoning directly.

## Rules
- Avoid redundant caching if it lives in standard `system/docs/`.
- Docs acts as "static" but highly-available context.
