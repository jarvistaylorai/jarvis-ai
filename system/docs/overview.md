# Overview

Welcome to the Documentation / Knowledge System for Jarvis. This system stores structured documents via a filesystem architecture (`/system/docs`), acting as a permanent intelligence layer.

It is designed for:
- **Context Injection**: Jarvis reads instructions natively.
- **Decision Consistency**: Maintains a single source of truth over time.

Docs are categorized and tagged for fast retrieval and future semantic search.
